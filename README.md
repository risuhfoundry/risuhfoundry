I'm a **full-stack engineer with hardware roots** — which means I think in systems,
end to end, from the microcontroller reading a sensor all the way to the cloud dashboard
visualising its data in real time.

My work sits at the intersection of **embedded systems**, **IoT infrastructure**, and
**AI-integrated products**. I care about the whole chain — clean firmware, reliable protocols,
solid APIs, and interfaces people actually want to use.

> `// idea → circuit → code → product`
